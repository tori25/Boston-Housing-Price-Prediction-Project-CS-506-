Dataset shape: (132611, 53)

Evaluation: 5-fold cross-validation on training set (mean ± std)

Story: linear baseline → regularization → non-linear → distance-based

Ridge Regression           MAE=   193,191±  2,008  RMSE=   308,208± 22,730  R²=0.6719±0.0532

Lasso Regression           MAE=   191,417±  1,939  RMSE=   307,933± 21,277  R²=0.6728±0.0497
  → Adds L1 penalty — forces some coefficients to zero (implicit feature selection).

Decision Tree              MAE=   233,756±    868  RMSE=   359,091±  1,518  R²=0.5574±0.0104
  → Splits on single features. Captures non-linearity but high variance.

Random Forest              MAE=   100,694±  1,004  RMSE=   174,169±  1,463  R²=0.8959±0.0029
  → 200 trees, each on a random feature subset. Reduces variance via averaging.

KNN (k=5)                  MAE=   112,013±  1,702  RMSE=   193,807±  3,482  R²=0.8711±0.0042
  → Predicts from the 5 nearest properties by Euclidean distance (k tuned by 5-fold CV).
